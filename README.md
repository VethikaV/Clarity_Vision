# Clarity Vision 🧠⚕️

Clarity Vision is an AI-powered healthcare support system designed to analyze medical data and assist in early detection of cancer and autoimmune diseases. The system integrates machine learning, deep learning, and OCR technologies to process medical images, reports, and clinical data.

---

## 🚀 Features

- 🩸 **Blood Cancer Detection**
  - Classifies blood cell images into:
    - Early Pre-B
    - Pre-B
    - Pro-B
    - Benign
  - Uses Convolutional Neural Networks (CNN)

- 📄 **Medical Report Analysis (OCR)**
  - Extracts text from scanned reports and PDFs
  - Cleans and processes unstructured medical data

- 🤖 **Disease Prediction**
  - Predicts cancer risk and autoimmune diseases
  - Models used:
    - Random Forest
    - Support Vector Machine (SVM)
    - Logistic Regression
    - XGBoost

- 🌐 **Multilingual Support**
  - Translates extracted medical text into multiple languages

- 💡 **Clinical Recommendations**
  - Provides:
    - Treatment suggestions
    - Confirmatory tests
    - General health advice

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Machine Learning:** Scikit-learn, XGBoost  
- **Deep Learning:** CNN (TensorFlow / PyTorch)  
- **OCR:** Tesseract OCR  
- **NLP & Translation:** Deep Translator  
- **Libraries:** NumPy, OpenCV, PIL
